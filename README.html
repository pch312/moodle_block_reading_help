Accessiblity Block

Description

This block allows you to modify the presentation of the text to make it easier to read for the visually impaired, the dyslexic...

Use

This block will allow you to modify:

  • The background color of the site
  • Text color
  • Text size
  • Line spacing
  • The space between characters
  • The space between the words
  • The typeface (Helvetica, Verdana, Comic, Dyslexic)
  • Font style (bold, italic)
The registration allows the user to find the same environment during his next connections.
Resetting removes any changes to the presentation.

Install

  • Download plugin
    https://www.chataigner.com/moodle/accessibilite/accessibilite.zip or https://github.com/pch312/moodle_bloc-accessibilite
  • Install it on your Moodle platform
  • Add additional HTML
    Inthe menu "Site administration" -> "Appearance" -> "Additional HTML",
    Add this texte in the field "Within HEAD" :
    <link rel="stylesheet" href="/blocks/accessibilite/font/comic-sans-ms.css">
    <link rel="stylesheet" href="/blocks/accessibilite/font/opendyslexic.css">
  • Add the block to the dashboard
