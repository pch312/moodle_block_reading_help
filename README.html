Reading Help Block

Description

This block allows you to read the text by voice and display a text mask

Use

2 buttons :

  • Speech synthesis, click on this button then select a text, it will be read.
  • Text mask, click on this button and a mask will be displayed to focus on a text area

Install

  • Download plugin
    https://www.chataigner.com/moodle/reading_help/reading_help.zip or https://github.com/pch312/moodle_bloc_reading_help
  • Install it on your Moodle platform
  • Add the block to the courses
