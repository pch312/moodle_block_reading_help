Bloc Accessiblite

Description

Ce bloc permet de modifier la présentation du texte pour faciliter la lecture pour les mal-voyants, les dyslexiques...

Fonctionnement

Ce bloc va permettre de modifier :

  • La couleur du fond du site
  • La couleur du texte
  • La taille du texte
  • L’interligne
  • L’espace entre les caractères
  • L’espace entre les mots
  • La police de caractères (Helvetica, Verdana, Comic, Dyslexic)
  • Le style des caractères (gras, italique)
    • L’enregistrement permet à l’utilisateur de retrouver le même environnement lors de ses prochaines connexions.
    La remise à zéro supprime toutes modifications de la présentation.

    Installation

    • Télécharger le plugin
      https://www.chataigner.com/moodle/accessibilite/accessibilite.zip ou https://github.com/pch312/moodle_bloc-accessibilite
    • Installer le sur votre plateforme Moodle
    • Ajout d'HTML additionnel
      Dans le menu "administration" -> "Présentation" -> "HTML Additionnel",
      ajouter ce texte dans le champ HEAD :
      <link rel="stylesheet" href="/blocks/accessibilite/font/comic-sans-ms.css">
      <link rel="stylesheet" href="/blocks/accessibilite/font/opendyslexic.css">
    • Ajouter le bloc sur le tableau de bord
