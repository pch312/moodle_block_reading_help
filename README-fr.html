Bloc aide à la lecture

Description

Ce bloc permet de lire le texte en vocal et d'afficher un masque de texte

Fonctionnement

2 boutons :

  • Synthèse vocale, cliquer sur ce bouton puis sélectionner un texte, il sera lu.
  • Masque de texte, cliquer sur ce bouton et un masque s'affichera pour vous concentrer sur une zone de texte

    • Installation

    • Télécharger le plugin
      https://www.chataigner.com/moodle/reading_help/reading_help.zip ou https://github.com/pch312/moodle_bloc_reading_help
    • Installer le sur votre plateforme Moodle
    • Ajouter le bloc sur les pages des cours
